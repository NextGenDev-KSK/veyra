# Bundled fonts

All three families are licensed under the **SIL Open Font License 1.1** (OFL),
which permits bundling and redistribution in GPLv3 software.

| File | Family | Role | Source |
|---|---|---|---|
| `Orbitron.ttf` | Orbitron (variable) | Display / wordmark | google/fonts `ofl/orbitron` |
| `Inter.ttf` | Inter (variable) | Body / UI text | google/fonts `ofl/inter` |
| `JetBrainsMono.ttf` | JetBrains Mono (variable) | Numeric readouts | google/fonts `ofl/jetbrainsmono` |

**Note on the display font:** the design spec calls for *Nasalization*, which is
only free for personal use and is **not** safe to redistribute under GPLv3. Per
the design doc's own guidance we substitute **Orbitron** (OFL), which has the
same geometric-futurist character. Update `Fonts.*` if a properly-licensed
display face is obtained.
